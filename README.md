# 📚 Gestión de Prácticas de Alumnos

![Build Status](https://img.shields.io/badge/status-en%20desarrollo-yellow) ![License](https://img.shields.io/badge/license-MIT-blue)

Esta es una aplicación web diseñada para la gestión de prácticas de los alumnos. Permite administrar y organizar las prácticas de manera eficiente. 

Desarrollada con **HTML, CSS, JavaScript, Node.js y Express.js**, esta aplicación aún está en fase de desarrollo, ya que falta implementar la gestión de bases de datos y la creación de APIs para el acceso a los datos.

---

## ✨ Funcionalidades

- 📌 **Registro y gestión de prácticas:** Los alumnos pueden visualizar y registrar sus prácticas.
- 👨‍🏫 **Administración de prácticas:** Los administradores podrán gestionar las prácticas de los alumnos.
- 🌐 **Interfaz web intuitiva:** Construida con tecnologías modernas para una mejor experiencia de usuario.
- 🔄 **Expansible:** Se planea la integración con bases de datos y APIs en el futuro.

## 🔧 Requisitos del Sistema

Este proyecto requiere un entorno con Node.js y un servidor compatible.

- **Node.js v14 o superior**
- **Express.js**
- **HTML, CSS y JavaScript**

---

## ℹ️ Instalación y Configuración

Sigue estos pasos para instalar y probar la aplicación:

### 1️⃣ Clonar el repositorio:
```bash
git clone https://github.com/AdrianMorenoGomez/GestionPracticas.git
```

### 2️⃣ Instalar dependencias:
```bash
cd GestionPracticas
npm install
```

### 3️⃣ Ejecutar la aplicación:
```bash
npm start
```

### 4️⃣ Acceder a la aplicación:
- Abre tu navegador y dirígete a:
  ```
  http://localhost:3000
  ```
